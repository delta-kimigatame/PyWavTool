# PyWavTool

### これは何?
* 飴屋／菖蒲氏によって公開されている、Windows向けに作成された歌声合成ソフトウェア「UTAU」に同梱されている、wavファイル結合用ソフトwavtool.exeの互換プロジェクトです。

    UTAU公式サイト(http://utau2008.web.fc2.com/)

* wavファイルを読み込んで、.whdと.datを生成・更新します。
* このプロジェクト単体ではあまり意味がありません。
