class dat.Dat(input: str)¶

ベースクラス: object

wavファイルのヘッダ以外のデータ部分を扱います。

data¶

wavファイルのデータ列

Type

np.ndarray

addframe(data: numpy.ndarray, ove: float, samplewidth: int, framerate: int) → int¶
self._datの末尾にdataを追加します。
ove(ms)分のデータは、self._dataとdataを加算します。
パラメータ

  • data (np.ndarray of float64) -- 書き込みするwavのデータ。1で正規化されている。

  • ove (float) -- 既存のframeにかぶせる長さ(ms)

  • samplewidth (int) -- wavのビット深度

  • framerate (int) -- wavのサンプル周波数

戻り値

nframes -- 追加したフレーム数

戻り値の型

int

addframeAndWrite(data: numpy.ndarray, ove: float, samplewidth: int, framerate: int, output: str) → int¶
outputのパスを開き、dataを追加します。
ove(ms)分のデータは、outputから読み込んだデータとdataを加算します。
パラメータ

  • data (np.ndarray) -- 書き込みするwavのデータ。1で正規化されている。

  • ove (float) -- 既存のframeにかぶせる長さ(ms)

  • samplewidth (int) -- wavのビット深度

  • framerate (int) -- wavのサンプル周波数

  • output (str) -- datファイルのパス。存在する場合上書きされます。

戻り値

nframes -- 追加したフレーム数

戻り値の型

int
