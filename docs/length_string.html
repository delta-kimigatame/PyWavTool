length_string module¶

length_string.str2float(length: str) → float¶

UTAUのLength文字列をfloatに変換します。

パラメータ

length (str) -- length文字列

戻り値

length

戻り値の型

float

メモ

lengthは以下のいずれかの形で与えられます。
tick@tempo
tick@tempo+delta
tick@tempo-delta
戻り値の計算は以下の通りです。
1拍あたりのms = 60*1000 / tempo
1tickあたりのms = 1拍あたりのms / 480
length = 1tickあたりのms * tick +(-) delta
